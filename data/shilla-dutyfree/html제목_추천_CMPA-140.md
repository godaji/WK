## 배경
현재 제목은 `면세점 위스키 추천` — 일반적이고 페이지의 진짜 차별점(656종을 가성비순 분석 +
「진짜 싼 것 vs 함정」 + 수상작/스토리)을 못 살린다. 과장 광고 톤을 싫어하는 결정권자
(공돌이·INTJ/ENTJ·가성비 선호)에게 **'추천'보다 '선별/검증'을 약속**하는 제목이 신뢰감을 준다.

- **`<h1>`** : `🥃 면세점 위스키, 진짜 싼 것 vs 함정`

- **인덱스 라벨** : `🥃 면세 위스키, 진짜 싼 것만 골랐다 (신라면세)`

이유: '추천'(누구나 하는 말) 대신 **'골랐다'(선별을 약속)** + **'함정'(가격분석의 날)** →
가성비·데이터 신뢰 + SEO 키워드("면세 위스키/가성비/가격") 전진 배치.

## 선택 가이드
- **A** = 페이지 핵심(가성비 분석)과 가장 정합. 결정권자 톤에 최적. → **채택**
- **B** = 클릭 자극(함정) 강함. 자극적이지만 약간 부정 프레임.
- **C** = 출장/공항 검색 유입(호치민 출장족 등 타깃)에 강점. 분석 깊이는 덜 드러남.
- **D** = 선물/스토리 탭을 띄우고 싶을 때. '추천' 단어가 다시 일반화 우려.

## HTML title 작성 규칙(재사용)
1. **길이** ~30자 내외(모바일/SERP 잘림 방지, 한글 기준 ≤ 35자 권장).
2. **핵심 키워드 전진**: "면세 위스키" 같은 검색어를 앞쪽에.
3. **브랜드 일관**: 끝에 `WK` 고정(— 또는 |로 구분).
4. **약속 동사**: '추천'(일반) 대신 '골랐다/분석/비교' 등 검증 뉘앙스.
5. `<title>`(SEO·탭) ↔ `<h1>`(페이지 후킹) ↔ `meta description`(SERP 요약) 톤 일치.
