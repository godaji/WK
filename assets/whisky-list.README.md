# whisky-list.csv — 위스키 마스터 리스트 (핵심 자산)

WK의 **크롤링/가격수집 대상 제품 마스터 리스트**. "어떤 위스키들의 가격을 추적할 것인가"의 정본(canonical list).
가격 시계열은 `data/whisky-prices/YYYY-MM.csv`가 담당하고, 이 파일은 **제품 식별·매칭 키**(한글명·영문명·종류·원산지)를 담는다.

- 생성: `python3 data/build_whisky_list.py` (재현 가능, 출처 데이터에서 큐레이션)
- 인코딩: UTF-8 (BOM) — Excel에서 한글 즉시 열림
- 규모: 위스키 88종 (high 45 / med 30 / low 13)

## 스키마
| 컬럼 | 설명 |
|---|---|
| `id` | 안정 식별자 (`w001`…). 크롤링 결과를 이 id로 조인 |
| `name_ko` | 한글 정규명 (국내 마트/데일리샷 매칭용) |
| `name_en` | 영문명 (해외 EC·글로벌 검색 매칭용). 일/대만/홍콩 크롤링의 핵심 키 |
| `brand` | 브랜드/증류소 |
| `category` | 싱글몰트 / 블렌디드 / 블렌디드몰트 / 버번 / 테네시 / 라이 / 아이리시 / 기타(국산 위스키류) 등 |
| `origin` | 원산지·지역 (예: 스코틀랜드-스페이사이드, 미국-켄터키, 일본, 대만) |
| `age` | 숙성연수 (NAS = 무연산) |
| `abv` | 도수(%) |
| `volume_ml` | 대표 용량 |
| `price_krw_low`/`price_krw_high` | 출처들에서 관측된 원화가 범위(참고용; 정밀가는 월간 CSV) |
| `channels` | 관측된 판매 채널 (트레이더스/코스트코/데일리샷/이마트 등) |
| `confidence` | high(정규 출처·이름 확실) / med(위스키픽 ASR 등 단일출처) / low(음차·OCR 추정) |
| `notes` | 변형 통합·OCR 추정·재확인 메모 |

## 데이터 출처 (모두 CMPA-1)
- `price-tracker` 문서 — 트레이더스/코스트코 가격 트래커
- 월간 CSV — `2026-03.csv`, `2026-04.csv`, `2026-05.csv`
- `2026-05_whiskeypick_traders_guwol.csv` — 위스키픽 ASR(트레이더스 구월점), 73행
- `research` 문서 — 가성비 위스키 리서치(영문명·종류·지역 출처)

## 정규화 규칙
- **위스키만** 포함. 보드카/진/테킬라/럼/리큐어/코냑은 제외(레미마틴·앱솔루트·봄베이·호세코에보·바카디·페로네·피치트리·바톤·다카라 망고 등 10종).
- 제품(브랜드+숙성/익스프레션) 단위로 dedup. **용량·패키지·지점 변형은 한 행으로 통합**(예: 조니워커 블랙 700ml/1.75L/12년 → 1행, 발베니 12년/더블우드 변형 → 1행).
- 위스키픽 ASR의 음차/자릿수 오류는 `confidence=low/med` + `notes`로 표기하고 **임의 보정하지 않음**. `(unverified)`·`미상` 표기 항목은 영문/브랜드 재확인 대상.
